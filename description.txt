Библиотеки:
os: для доступа к переменным окружения.
spacy: анализ текстов и извлечение зависимостей между словами.
svgwrite: создания SVG-файлов, используется для визуализации зависимостей слов.
dotenv: модуль для загрузки переменных окружения из файла .env.
telebot: библиотека для работы с Telegram Bot API.

Структура и функции:
1) ИМПОРТ БИБЛИОТЕК.
2) ЗАГРУЗКА ПЕРЕМЕННЫХ ОТКРУЖЕНИЯ.
Выполняется с помощью функции load_dotenv().
3) ПОЛУЧЕНИЕ ТОКЕНА
Используется os.getenv("API_TOKEN") для получения токена бота из переменной окружения.
4) ЗАГРУЗКА SPACY.
Модель en_core_web_sm загружается для выполнения анализа текста на английском языке.
5) ОБРАБОТЧИК СООБЩЕНИЙ.
Декоратор @bot.message_handler(func=lambda message: True) указывает, что функция handle_message будет вызываться для всех входящих сообщений.
6) АНАЛИЗ ТЕКСТА.
Производится с помощью handle_message.
7) СОЗДАНИЕ SVG-ФАЙЛА.
8) СОХРАНЕНИЕ SVG-ФАЙЛА.
9) ОТПРАВКА ФАЙЛА ПОЛЬЗОВАТЕЛЮ.
10) ЗАПУСК БОТА
С помощью bot.polling().
